# VXE
Smart contracts for the vxe.

# Installation
1. Ensure truffle is installed. More info here [Trufffle Installation](https://truffle.readthedocs.io/en/beta/getting_started/installation/)
2. Clone this repo
3. Do:
  - `npm install`
  - `make`
